Clam Antivirus
==============

## Installation

```
	sudo apt install clamav	
```

## Update Database

```
	sudo freshclam --show-progress	
```

## Scan Files

```
	clamscan <file>
```

```
	Options:
		-v	verbose
		-i	only print infected files
		-r	scan a directory recursively
```
