Matrix In Latex
===============

## Requirements

To use matrix in a Latex document include "amsmath" package.

## Plain Matrix

```
	$$\begin{matrx}	
		a & b//
		c & d
	\end{matrix}$$
```

## Parenthesis Matrix

```
	$$\begin{pmatrx}	
		a & b//
		c & d
	\end{pmatrix}$$
```

## Bracket Matrix

```
	$$\begin{bmatrx}	
		a & b//
		c & d
	\end{bmatrix}$$
```

## Braces Matrix

```
	$$\begin{Bmatrx}	
		a & b//
		c & d
	\end{Bmatrix}$$
```

## Pipe(Determinant) Matrix

```
	$$\begin{vmatrx}	
		a & b//
		c & d
	\end{vmatrix}$$
```

## Double Pipe(Determinant) Matrix

```
	$$\begin{vmatrx}	
		a & b//
		c & d
	\end{vmatrix}$$
```
