Wallabag on Koreader

Get a wallbag account on an wallbag instance1. Wallabagger is Pocket like free software service. It has readability feature built into it. It can be used to read readable webpages on kindle.

On Kindle

In KOreader topmenu select tools > wallabag > settings > configure wallabag server. Fill available fields. Now koreader is set to use wallabag.

On firefox

Install wallabagger2 addon on your firefox browser and configure it.
