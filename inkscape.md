Inkscape Howto
==============

Inkskape is a freesoftware for creating and editing vector graphics.


## Export To PNG

```
	inkscape -D -z --file=libtexts-tangles.svg --export-dpi=300 --export-png=libtexts-tangles.png
```
## Shortcut Keys
```
	+/-		Zoom in/zoom out
```

## Toolbar
```
	arrow		for selecting objects
```
